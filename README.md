# ماسح إيصالات الكسارات — AI Agent (نسخة ويب عامة)

أداة ويب مستقلة (بدون تنصيب) لأي حد يفتح الرابط: يرفع صورة/PDF لإيصال توريد مواد بناء،
وياخد ملف إكسيل فيه كل إيصال في صف واحد.

**مش OCR عادي** — بيستخدم نموذج ذكاء اصطناعي (GPT-4o Vision) يفهم المستند، يوحّد أسماء
الحقول المختلفة بين الكسارات، يتحقق حسابيًا إن الوزن الصافي = القائم − الفارغ، ويعلّم أي
إيصال فيه شك أو تعارض باللون الأصفر في الإكسيل.

## كيفية النشر على Vercel (مرة واحدة فقط)

1. على [vercel.com](https://vercel.com) اضغط **Add New → Project** واختر هذا الـ Repository
   (تم استيراده تلقائيًا لأن Vercel مربوط بحساب GitHub).
2. قبل الضغط على Deploy، افتح **Environment Variables** وأضف:
   - `OPENAI_API_KEY` = مفتاح OpenAI الخاص بك (يبدأ بـ `sk-...`)
3. اضغط **Deploy**. بعد دقيقة هيديك رابط دائم زي:
   `https://receipt-scanner-xxxx.vercel.app`
4. ابعت الرابط ده لأي حد — هيفتح الأداة في المتصفح مباشرة، من غير أي تنصيب.

> أي تحديث تعمله بعد كده على الكود في GitHub، Vercel هينشره تلقائيًا (Auto Deploy).

## البنية

```
index.html      — الواجهة (رفع الملفات + عرض النتائج + تحميل الإكسيل)
styles.css      — تصميم بألوان Sina Investment Group (أحمر/رمادي غامق)
script.js       — منطق العميل: تقسيم PDF، تصغير الصور، استدعاء /api/analyze، تصدير Excel
api/analyze.js  — Serverless Function تستدعي OpenAI Vision بمفتاح مخزّن آمن على السيرفر
api/prompt-text.js — نص البرومبت المعتمد (نفس الحقول والقواعد في نسخة سطح المكتب)
```

## ملاحظات أمان

- مفتاح OpenAI **لا يظهر أبدًا في المتصفح** — يتم استخدامه فقط داخل `api/analyze.js` على سيرفر Vercel.
- الصور بترفع وقت التحليل فقط ولا يتم تخزينها بشكل دائم على السيرفر.
- الرابط عام: أي حد معاه الرابط يقدر يستخدم الأداة ويستهلك من رصيد الـ API بتاعك. لو عايز
  تحميه بكلمة مرور بسيطة، قولّي وهضيفها.
